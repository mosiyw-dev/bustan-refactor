# 🌱 سامانه دانشگاهی بوستان - پروژه تمرینی

این ریپو شامل پیاده‌سازی تمرینی بخش‌هایی از سامانه‌ی دانشگاهی بوستان است که با هدف یادگیری و تسلط بیشتر بر توسعه‌ی فرانت‌اند طراحی شده. هدف اصلی، شبیه‌سازی برخی ماژول‌ها یا صفحات کاربردی این سامانه است.

## 🛠️ تکنولوژی‌های استفاده‌شده

- [Next.js](https://nextjs.org/)
- [React](https://reactjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [TypeScript](https://www.typescriptlang.org/)
- [shadcn/ui](https://ui.shadcn.com/)
